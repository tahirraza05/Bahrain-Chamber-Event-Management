# Open App in Cursor Browser

## Quick Access
**URL:** http://localhost:4200

## Method 1: Command Palette
1. Press `Ctrl+Shift+P` (Windows/Linux) or `Cmd+Shift+P` (Mac)
2. Type: `Simple Browser: Show`
3. Press Enter
4. Enter: `http://localhost:4200`

## Method 2: Sidebar Browser Icon
1. Look for the browser/globe icon in Cursor's left sidebar
2. Click it
3. Enter: `http://localhost:4200`

## Method 3: Direct Link
If Cursor supports it, you can also:
- Right-click on this file
- Select "Open in Browser" or similar option

## Note
Make sure the Angular development server is running:
- Check the PowerShell window that's running `npm start`
- Or run: `npm start` in the terminal
