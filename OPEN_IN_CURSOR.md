# How to Open App in Cursor Browser

## Quick Steps:

1. **Make sure the server is running:**
   - Check if you see "Angular Live Development Server is listening on localhost:4200" in your CMD window
   - If not, run: `npm start` in the project directory

2. **Open Cursor's Simple Browser:**
   - Press `Ctrl+Shift+P` (Windows/Linux) or `Cmd+Shift+P` (Mac)
   - Type: `Simple Browser: Show`
   - Press Enter
   - Enter URL: `http://localhost:4200`
   - Press Enter

3. **Alternative - Use Command Palette:**
   - Press `Ctrl+Shift+P`
   - Type: `Simple Browser`
   - Select "Simple Browser: Show"
   - Enter: `http://localhost:4200`

## If Server is Not Running:

1. Open terminal in Cursor (Ctrl+`)
2. Navigate to project: `cd C:\Users\TAHIR\bahrain-chamber-app`
3. Run: `npm start`
4. Wait for "Compiled successfully"
5. Then open Simple Browser as above

## Direct URL:
http://localhost:4200
